# PWA 图标

请在此目录放置以下尺寸的 PNG 图标：

| 文件名           | 尺寸    | 必需    |
| ---------------- | ------- | ------- |
| icon-192x192.png | 192×192 | ✅ 必需 |
| icon-512x512.png | 512×512 | ✅ 必需 |

## 设计要求

- 使用元思想品牌标识
- 背景色建议：#F5F0E8（暖米色）或透明
- 图标主体居中，留有适当边距
- maskable 图标需要在安全区域内（圆形裁切不丢失内容）

## 生成工具推荐

- [PWA Asset Generator](https://github.com/nickvdyck/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Figma 导出
