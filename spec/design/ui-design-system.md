## 一、设计理念

### 1.1 核心价值

| 关键词     | 说明                                             |
| ---------- | ------------------------------------------------ |
| **回归**   | 回到当下，回到自己                               |
| **温暖**   | 琥珀金的智慧之光，给予温暖引导                   |
| **自然**   | 鼠尾草绿的自然生长，疗愈力量                     |
| **沉浸**   | 全屏背景 + 呼吸动画，创造沉浸体验                |
| **纪元感** | 古纸底色 + 节气时辰 + 宋体字，古卷展开的时间质感 |

### 1.2 视觉风格

- **简约第一**：大量留白，克制装饰
- **暖色柔和**：琥珀金 + 鼠尾草绿 + 古纸底色
- **中国元素点睛**：节气时辰、宋体字已实现；印章标记规划中
- **整体保持现代简约**，中国元素为调味而非主体

### 1.3 中国元素策略

| 元素     | 状态      | 说明                         |
| -------- | --------- | ---------------------------- |
| 节气时辰 | ✅ 已实现 | 中国哲学时间体系替代现代日期 |
| 宋体字   | ✅ 已实现 | Noto Serif SC 用于标题和金句 |
| 印章标记 | 📋 规划中 | 品牌视觉识别                 |

**原则**：中国元素为点睛之笔，不喧宾夺主。

## 五、阴影与毛玻璃

### 5.1 阴影系统

```css
--shadow-xs: 0 1px 4px rgba(0, 0, 0, 0.06); /* 微阴影 */
--shadow-sm: 0 2px 8px rgba(0, 0, 0, 0.08); /* 标准卡片 */
--shadow-md: 0 4px 16px rgba(0, 0, 0, 0.12); /* hover 状态 */
--shadow-lg: 0 8px 24px rgba(0, 0, 0, 0.15); /* 浮起效果 */
--shadow-xl: 0 12px 32px rgba(0, 0, 0, 0.18); /* 模态框 */
```

**核心原则**：用阴影营造立体感，**不用明显边框**。

### 5.2 毛玻璃效果

```css
--blur-light: blur(12px);
--blur-medium: blur(20px);
--blur-heavy: blur(32px);
--blur-extra: blur(40px);
```

**兼容性注意**：

- `backdrop-filter` 在小程序中支持有限
- 底部导航栏已改用纯半透明背景替代
- 其他毛玻璃区域转换时用半透明纯色替代

## 六、750px 响应式系统

### 6.1 设计基准

- **设计稿宽度**：750px（微信小程序标准）
- **换算关系**：在 750px 设计稿中，1px ≈ 2rpx
- **Web 端实现**：使用 CSS 变量 `--rpx` 实现响应式适配

### 6.2 核心 CSS 变量

```css
--rpx: calc(100vw / 750); /* 小屏使用 vw */
/* 大屏（>750px）时固定为 1px，防止元素过大 */
```

### 6.3 使用方式

```tsx
// 方式1：使用 rpx 工具函数（推荐）
import { rpx } from '../config/styles';
<div style={{ padding: rpx(48) }}>  {/* 48rpx ≈ 24px */}

// 方式2：直接使用 CSS 变量
<div style={{ padding: 'calc(var(--rpx) * 48)' }}>
```

### 6.4 最大宽度限制

```css
.app-container {
  max-width: 750px;
  margin: 0 auto;
}
```

## 七、动画原则

### 7.1 过渡时间

| 名称   | 时长  | 使用场景   |
| ------ | ----- | ---------- |
| fast   | 150ms | 按钮 hover |
| normal | 250ms | 标准过渡   |
| slow   | 400ms | 大元素过渡 |

### 7.2 品牌呼吸动画

| 组件            | 周期  | 说明                        |
| --------------- | ----- | --------------------------- |
| BreathingCircle | 8秒   | 4秒吸 + 4秒呼，三层独立动画 |
| GuideButton     | 3.5秒 | opacity 脉动                |
| ScrollHint      | 2秒   | translateY + opacity        |

### 7.3 动画原则

- 动画要克制，不过度
- 所有动画感觉"自然"、"柔和"
- 呼吸动画是品牌标志性交互

## 八、阅读方向策略

| 元素类型   | 方向   | 说明                       |
| ---------- | ------ | -------------------------- |
| 功能性元素 | 左→右  | 卡片入口、导航栏、按钮     |
| 意境性元素 | 右→左  | 节气竖排、装饰文字         |
| 循环性元素 | 无方向 | 呼吸圆环，通过动画节奏引导 |

**三卡片顺序**：徐→止→定（左→右），由动入静的渐修路径。

## 十、设计检查清单

### 新建组件时

- [ ] 使用 `config/styles.ts` 中的共享常量
- [ ] 文本/图片放在 config 层而非硬编码
- [ ] 圆角使用 CSS 变量（`--radius-*`）
- [ ] 用阴影而非边框
- [ ] 注释清晰（用途、Props、使用场景）

### 新建页面时

- [ ] 基于 750px 设计基准
- [ ] 页面只负责组装，不含硬编码数据
- [ ] 宋体字只用于标题/金句
