# 项目说明文档

## 项目背景

**元思想**是一个身心成长平台应用，使用 React + Tailwind CSS 开发原型，最终目标转换为 TARO 框架（SCSS），适配微信小程序和 H5。

### 重要原则

- **原创设计**：融合现代简约与中国传统美学，所有实现均为原创
- **最终目标**：转换为 TARO + SCSS 框架，适配微信小程序和 H5
- **组件复用**：任何出现 2 次及以上的元素都应抽离为公共组件
- **配置分离**：文本、图片、图标等配置性内容独立于组件，放在 `config/` 目录统一管理

## 设计风格

### 核心视觉语言

- **简约柔和**：大量留白，克制装饰
- **暖色体系**：琥珀金 `#C49A6C` + 鼠尾草绿 `#8BAA7D`
- **古纸底色**：`#F0E4CE → #F5E8D2`，营造"古卷展开"的纪元感（首页/文章页）
- **宇宙星空深色**：`#0B1120 → #111D35`，人类手册模块专属（听书/学习场景）
- **中国元素"两处点睛 + 一处规划"**：节气时辰、宋体字已实现；印章标记规划中
- **整体保持现代简约**，中国元素为点睛而非主体
- **模块配色差异化**：首页古纸暖色 / 人类手册深空蓝 / 呼吸页浅绿，避免全站单调

### 关键设计原则

1. **阴影代替边框**：用 box-shadow 营造立体感，不用明显 border
2. **宋体标题**：标题和金句使用 `'Noto Serif SC', serif`
3. **750px 宽度基准**：所有尺寸基于 750px 设计稿，使用 `--rpx` 响应式单位
4. **滚动条隐藏**：全局隐藏滚动条，保留滚动功能
5. **白色文字可读性**：Hero 区白色文字加三层 text-shadow 保证在亮背景上可读
6. **毛玻璃谨慎使用**：小程序兼容性有限，优先使用半透明纯色

## 页面架构

### 首页（Home）— 固定首屏 + 内容上滑覆盖

**交互架构**：
- Hero 区 `position: fixed` 固定不动，始终停留原位
- 内容页从底部自然上滑覆盖 Hero，圆角透出背景图
- Header 在内容页接近顶部时过渡为绢轴色吸顶样式，与内容页背景融为一体
- 向下滑动回退时，内容页下移，Header 恢复透明 Hero 样式

**隐喻**：向上滑动 = 揭开新的一页、向内探索

#### Hero 区（首屏 100vh，固定不动）
- 全屏自然风景背景图（高空俯瞰绿色大地河谷）
- 背景渐变遮罩：深琥珀色 `rgba(30,20,10,...)` 配合暖色体系
- 右上：24节气 + 十二时辰 + 诗句（古卷竖排）
- 左上：用户头像
- 中央：呼吸引导圆环（收｜清，吸气高亮「收」/呼气高亮「清」，三层呼吸动画）
- 下方：「问道」按钮（弧线包裹+呼吸动画）+ 徐/止/定三张玻璃卡片（左→右递进：慢→停→静，纵向布局：文字在上图标在下）+ 觉知滚动提示
- Header 吸顶阈值：滚动 > 90vh（内容页顶部接近 Header 区域时触发）

#### 内容区（上滑覆盖 Hero）
- 顶部圆角 28rpx，上滑时圆角透出固定 Hero 背景图，呈现"卡片浮在风景上"效果
- 绢轴色渐变背景 `#F0E4CE → #F5E8D2`
- 内容依次为：
  1. 今日之光（DailyWisdomCard）— 语录卡片
  2. 回到此刻，感知呼吸（FeaturedCard）— 精选推荐大卡片
  3. 今日频率指南（SectionBlock，2列宽图）
  4. 感知入门（SectionBlock，3列方图）
  5. 明镜之声（SectionBlock，2列宽图）
  6. 感知者的声音（SectionBlock，3列方图）
  7. 正在发生的光点（SectionBlock，3列方图）
  8. 官方公告（AnnouncementCard）

### 阅读方向策略

首页混合使用传统与现代阅读方向，遵循以下原则：
- **功能性入口**（卡片、导航栏）：左→右现代方向，优先用户直觉
- **意境性装饰**（节气竖排）：右→左传统方向，营造古卷氛围
- **循环性元素**（呼吸圆环收/清）：无固定方向，通过动画节奏引导认知

### 音频资源
- **音频合成器**：`/src/app/utils/audio-generator.ts` — Web Audio API 合
- **零外部依赖**：所有音频由浏览器实时合成（OfflineAudioContext → WAV Blob）
- **零版权风险**：代码生成的纯音频，非采样素材
- **7 种音频**：
  | 类型 | 场景 | 合成技术 |
  |------|------|---------|
  | rain | 呼吸页 · 细雨 | 带通噪音 + 振幅调制 |
  | ocean | 呼吸页 · 海浪 | 低通噪音 + 正弦调制（10秒/浪） |
  | forest | 呼吸页 · 山林 | 粉噪 + 高频鸟鸣脉冲 |
  | wind | 呼吸页 · 清风 | 带通噪音 + 极慢调制 |
  | xu | 感知引导 · 徐 | 174Hz 索尔菲吉奥 + 泛音 |
  | zhi | 感知引导 · 止 | 136.1Hz OM 频率 + 和弦泛音 |
  | ding | 感知引导 · 定 | 111Hz 冥想共振 + 双耳节拍 |
- **升级为真实音频**：将 config 中 audioSrc 改为 CDN URL 即可，合成器自动跳过
- **推荐免费音频源**：Pixabay Music、Freesound.org、Free Music Archive、Mixkit

### TARO 转换 — 音频相关
- Web Audio API 在小程序中不可用
- 需预录制 mp3 文件 + `wx.createInnerAudioContext()` 替代
- 环境音效建议从 Freesound.org（CC0 License）下载替代
- `resolveAudioUrl()` 不再需要，直接使用文件路径
