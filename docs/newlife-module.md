# 新人生之路模块 - 开发文档

## 模块定位

「新人生之路」是元思想平台的第三个底部导航 Tab，融合**课程学习**和**圈子社区**两大场景。
参照小鹅通「新人生之路圈子」的交互体验，保持用户习惯友好。

## 核心功能（MVP v1.0）

### 已实现
- **主页**（NewLifePath.tsx）：Banner + 分类 Tab（推荐/课程/练习/互动）
  - 推荐 Tab：继续学习卡片 + 圈子入口 + 全部课程
  - 课程/练习 Tab：按子分类分组的课程列表
  - 互动 Tab：跳转到圈子社区
- **圈子社区**（CirclePage.tsx）：参照小鹅通圈子布局
  - 圈子信息头部（封面、名称、简介、动态数/成员数、排行榜入口）
  - 官方公告条
  - 功能导航（课程/成员/更多/我的 — MVP 阶段 Toast 占位）
  - 内容分类 Tab（全部/精选/圈主/问答/官方通告/闭关·元振动）
  - 帖子 Feed 流：
    - 头像、用户名、角色标签（圈主/嘉宾）、用户等级（Lv）
    - 话题标签（#元振动分享 等，蓝色链接样式）
    - 帖子内容
    - 点赞用户名列表（"心向阳光，香云… 觉得很赞"）
    - 热门评论（用户名：评论内容）
    - 底部互动栏（点赞数/评论数/分享数，三等分布局）
    - 右上角下拉箭头
  - **发帖浮动按钮**（fixed 定位，蓝色圆形，不跟随滚动）
    - 点击弹出底部菜单：发动态 / 去提问
  - **发表动态页**（全屏）：
    - 顶部返回+标题
    - 全屏文本输入区
    - 底部工具栏：# 标签 / 图片 / 视频 / 表情 / 更多 + 发送按钮
  - 问答 Tab 专属答主面板
- **底部导航三 Tab 全互通**：首页 ↔ 人类手册 ↔ 新人生之路

### MVP 阶段简化
- 课程点击为 Toast 占位（无课程详情页）
- 圈子功能导航（课程/成员/更多/我的）为 Toast 占位
- 点赞/评论/分享/发帖为 Toast 占位
- 帖子为静态 mock 数据，无实时刷新
- 无用户鉴权，所有内容公开可见

```

  newlife-data.ts      — 分类/课程/圈子/帖子 mock 数据 + 工具函数
```

## 路由

| 路由 | 页面 | 入口 |
|------|------|------|
| `newlife` | 课程+社区主页 | 底部导航 Tab 3 |
| `circle` | 圈子社区 | 互动 Tab / 圈子入口卡片 |

返回路径：圈子 → 新人生之路主页

## 配色策略

- **主页**：清新现代白底 `#FAFAF7` + 暖琥珀金强调色 `#C49A6C`
- **圈子头部**：温暖棕色 `#A07D55`（对标小鹅通风格）
- **帖子区域**：纯白底 `#FFFFFF`，间隔用浅灰 `#F9F8F5`
- 与首页古纸色、人类手册深空蓝形成三方差异化

## 圈子内容分类（对标小鹅通）

| Tab | 说明 | 内容来源 |
|-----|------|---------|
| 全部 | 所有帖子 | 全量 Feed |
| 精选 | 管理员标记的精华帖 | `isFeatured: true` |
| 圈主 | 圈主发布的内容 | `role: "owner"` |
| 问答 | 问答帖 + 答主面板 | `category: "qa"` |
| 官方通告 | 官方公告帖 | `category: "announcement"` |
| 闭关/元振动 | 元振动分享、闭关心得 | `category: "retreat"` |

## 角色体系

| 角色 | 标签 | 颜色 | 说明 |
|------|------|------|------|
| owner | 圈主 | 琥珀金 | 社区管理者 |
| guest | 嘉宾 | 鼠尾草绿 | 官方答疑/特邀嘉宾 |
| member | （无标签） | — | 普通成员 |

## 与小鹅通的差异

| 功能 | 小鹅通 | 元思想 MVP |
|------|--------|-----------|
| 圈子 Feed | 实时加载 | 静态 mock 数据 |
| 发帖 | 支持图文 | Toast 占位 |
| 互动 | 点赞/评论/转发 | Toast 占位 |
| 课程播放 | 视频/音频 | 跳转到人类手册播放器（后续） |
| 成员管理 | 完整体系 | Toast 占位 |
| 搜索/筛选 | 多维度 | Toast 占位 |

## 设计注意事项

- Banner 区域使用秋季暖色调大图，符合「新人生之路」的温暖基调
- 帖子 Feed 流采用小鹅通经典布局：头像+用户名+角色标签+时间+内容+互动栏
- 发帖按钮为浮动圆形按钮（右下角），参照小鹅通风格
- 圈子信息头部使用温暖棕色背景，与白色帖子区形成视觉分层
- 所有尺寸使用 `rpx()` 响应式单位
- 宋体（Noto Serif SC）仅用于标题和圈子名称
