## 📚 一、人类手册模块

### 定位

> 元思想的思想体系核心载体，十部完整的电子书 + 配套音频课程

### 模块结构

```
人类手册
│
├── 📖 电子书阅读
│   ├── 十部电子书列表
│   │   ├── 第一部：[书名]
│   │   ├── 第二部：[书名]
│   │   ├── ...
│   │   └── 第十部：[书名]
│   │
│   ├── 阅读器功能
│   │   ├── 章节导航
│   │   ├── 阅读进度记录
│   │   ├── 书签功能
│   │   ├── 字体大小调节（适配中老年用户）
│   │   └── 夜间/日间模式
│   │
│   └── 阅读历史
│       └── 继续阅读上次位置
│
├── 🎧 音频课程
│   ├── 课程列表（与电子书章节对应）
│   ├── 音频播放器
│   │   ├── 播放/暂停
│   │   ├── 进度条拖动
│   │   ├── 倍速播放（0.5x/1x/1.5x/2x）
│   │   ├── 上一章/下一章
│   │   ├── 顺序播放/单曲循环
│   │   └── 后台播放
│   │
│   └── 学习进度
│       ├── 已学习时长统计
│       └── 章节完成状态
│
└── 🛒 书籍购买入口（商城集成）
    ├── 实体书购买
    └── 电子书付费解锁（如有）
```

### 设计建议

- **电子书与音频深度绑定**：同一章节可切换阅读/收听模式
- **进度跨设备同步**：基于用户身份，多端同步学习进度
- **商城入口内嵌**：在人类手册模块内直接提供购买入口，减少跳转

## 🛤️ 二、新人生之路模块

### 定位

> 学员学习与互动的主阵地，取代小鹅通（鹅圈子），核心是音频课程 + 社区互动

### 模块结构

```
新人生之路
│
├── 🎧 课程学习（核心功能）
│   ├── 课程列表
│   │   ├── 课程封面
│   │   ├── 课程简介
│   │   ├── 章节数量
│   │   └── 学习进度百分比
│   │
│   ├── 课程详情页
│   │   ├── 章节列表
│   │   ├── 当前学习位置标记
│   │   └── 继续学习按钮
│   │
│   └── 音频播放器（同人类手册规格）
│
├── 💬 圈子（社区互动）
│   ├── 动态发布
│   │   ├── 文字动态
│   │   ├── 图片动态
│   │   └── 音频/语音动态（可选）
│   │
│   ├── 动态流浏览
│   │   ├── 最新动态
│   │   ├── 热门动态
│   │   └── 关注的人
│   │
│   └── 互动功能
│       ├── 点赞
│       ├── 评论
│       └── 分享
│
├── 📢 动态/公告
│   ├── 官方公告
│   ├── 课程更新通知
│   └── 活动预告
│
└── 🛒 课程购买入口（商城集成）
    └── 课程付费解锁
```

### 设计建议

- **课程为主，社区为辅**：首屏突出课程学习入口，圈子作为二级功能
- **丝滑的学习体验**：比小鹅通更流畅的音频播放和进度记录
- **圈子适度克制**：不做成热闹的社交平台，保持元思想的"清净"气质

│   ├── 地点信息
│   ├── 费用说明
│   └── 人数限制
│

### 定位

```

│   │   └── 流式响应显示
│   │

│   └── 觉醒层级提示

- **技术已就绪**：基于现有mingjing-fastapi-proxy后端

## 🔧 通用功能模块设计建议

### 问题：通用功能是集成到各模块，还是独立模块？

#### 分析

| 功能          | 建议方案                      | 理由                                           |
| ------------- | ----------------------------- | ---------------------------------------------- |
| **商城/支付** | 内嵌到各模块                  | 用户在哪里产生需求，就在哪里完成购买，减少跳转 |
| **圈子/动态** | 主要放在"新人生之路"          | 这是社区互动的主阵地；其他模块保持"静"         |
| **直播**      | 放在"新人生之路"              | 直播是课程延伸，与学习场景绑定                 |
| **消息通知**  | 全局底层能力                  | 各模块的通知统一收口                           |
| **用户中心**  | 独立入口（底部Tab或设置入口） | 个人信息、订单、学习记录等统一管理             |

### 推荐架构

```
┌─────────────────────────────────────────────────────────────────┐
│                        元思想平台                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌────────┐│
│  │人类手册  │ │新人生之路│ │感知训练营│ │元镜中心  │ │ 明镜   ││
│  │          │ │          │ │          │ │          │ │        ││
│  │ 📖电子书 │ │ 🎧课程   │ │ 📋介绍   │ │ 🏛️介绍  │ │ 💬对话 ││
│  │ 🎧音频   │ │ 💬圈子   │ │ 📅期次   │ │ 🧘项目   │ │        ││
│  │ 🛒购买   │ │ 📢动态   │ │ ✍️报名   │ │ 📅预约   │ │        ││
│  │          │ │ 📺直播   │ │ 🛒支付   │ │ 🛒支付   │ │        ││
│  │          │ │ 🛒购买   │ │          │ │          │ │        ││
│  └──────────┘ └──────────┘ └──────────┘ └──────────┘ └────────┘│
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                      通用底层能力                                │

## 🎨 设计原则提醒

基于先明老师指导，所有模块设计必须遵循：

| 要的             | 不要的           |
| ---------------- | ---------------- |
| 极简、透明、清澈 | 冥想风、疗愈风   |
| 碑感、明亮、穿透 | 小清新风、灵性风 |
| 中轴强、主轴清晰 | 柔软治愈产品感   |
| 镜面级入口感     | 内容流先行       |

> **核心定调**：不要做"疗愈空间"，要做"归位之镜"
